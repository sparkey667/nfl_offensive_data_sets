# nfl_offensive_data_sets

Hello there,
I this repository contains csv files of NFL teams offensive stats. Some of the stats include:
  * Kicking
  * Kick Returns
  * Team Stats
  * Offensive Line
  * Passing
  * Punting
  * Receiving
  * Rushing
  * Scoring
  * Total Yards
  
 Feel free to download these data sets, ask questions, and give feedback
